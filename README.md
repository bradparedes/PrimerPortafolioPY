# 🐍 Snake Game

Este es mi primer proyecto de videojuego desarrollado con **Python** utilizando la biblioteca **Tkinter**. Es una implementación del clásico juego de Snake, diseñado como parte de mi aprendizaje en desarrollo de software. 

## 🎮 Características
- Movimiento continuo de la serpiente, con controles direccionales.
- Crecimiento dinámico al consumir comida.
- Sistema de puntaje que incrementa al comer.
- Detecta colisiones con las paredes o el propio cuerpo para finalizar el juego.
- Fácil personalización de colores, tamaño y velocidad del juego.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python
- **Biblioteca gráfica:** Tkinter
- **Conceptos:** Programación orientada a objetos (POO), manejo de eventos y lógica de juegos.

## 🚀 Cómo Ejecutar el Proyecto
Sigue estos pasos para ejecutar el juego en tu máquina local:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/bradparedes/PrimerPortafolioPY.git
   ```

Navega a la carpeta del proyecto:
    ```
    cd PrimerPortafolioPY/SnakeGame
    ```
Asegúrate de tener Python instalado (versión 3.6 o superior).

Ejecuta el archivo principal:
python snake_game.py

🌟 Personalización
Puedes modificar las siguientes variables en el archivo snake_game.py para personalizar el juego:

GAME_WIDTH y GAME_HEIGHT: Define el tamaño del área de juego.

SPEED: Ajusta la velocidad de movimiento de la serpiente.

Colores: Cambia los colores de la serpiente (SNAKE_COLOR), la comida (FOOD_COLOR) y el fondo (BACKGROUND_COLOR).

BODY_PARTS: Cambia el tamaño inicial de la serpiente.


📚 Aprendizaje y Objetivos
A través de este proyecto, he aprendido:

Cómo crear interfaces gráficas con Tkinter.

Implementar lógica de juegos en Python.

# 📝 Bloc de Notas

Este es un proyecto simple de Bloc de Notas desarrollado en **Python** utilizando la biblioteca **Tkinter**. Es una herramienta básica pero funcional para tomar, editar y guardar notas.

## ✨ Características
- **Crear notas nuevas**: Comienza desde cero con una nueva nota en blanco.
- **Editar notas existentes**: Abre archivos de texto guardados y edítalos.
- **Guardar notas**: Guarda tus notas en archivos de texto para usarlas posteriormente.
- **Interfaz intuitiva**: Fácil de usar, con menús desplegables para todas las funciones principales.
- **Multiplataforma**: Compatible con sistemas Windows, Mac y Linux (siempre que Python esté instalado).

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python
- **Biblioteca gráfica:** Tkinter
- **Conceptos:** Manejo de eventos, entrada/salida de archivos, diseño de interfaces gráficas.

## 🚀 Cómo Ejecutar el Proyecto
1. Clona este repositorio:
   ```bash
   git clone https://github.com/bradparedes/PrimerPortafolioPY.git
   ```
   Navega a la carpeta del proyecto:
   ```bash
   cd PrimerPortafolioPY/BlokNote
   ```
Asegúrate de tener Python instalado (versión 3.6 o superior).

Ejecuta el archivo principal:
   ```bash
   python BlokNote.py
   ```
¡Comienza a escribir y guardar tus notas!

🌟 Personalización

# ❌ Tik Tak Tok

Este es un proyecto simple de **Tic Tac Toe (Tres en Raya)** desarrollado en **Python** utilizando la biblioteca **Tkinter**. Es un juego interactivo que permite a dos jugadores competir entre sí desde una misma interfaz gráfica.

## ✨ Características
- **Modo para dos jugadores:** Juega contra otra persona desde el mismo dispositivo.
- **Interfaz gráfica amigable:** El tablero es fácil de entender e interactuar.
- **Detección automática de ganadores:** El juego determina al ganador o si ocurre un empate.
- **Reinicio rápido:** Incluye un botón para reiniciar el juego sin cerrar la aplicación.
- **Diseño minimalista:** Limpio, intuitivo y fácil de usar.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python
- **Biblioteca gráfica:** Tkinter
- **Conceptos:** Lógica de juegos, manejo de eventos y diseño de interfaces gráficas.

## 🚀 Cómo Ejecutar el Proyecto
1. Clona este repositorio:
   ```bash
   git clone https://github.com/bradparedes/PrimerPortafolioPY.git
   ```
Navega a la carpeta del proyecto:
   ```bash
   python TikTakTok.py
   ```
¡Disfruta del juego con un amigo!

🌟 Personalización
Puedes cambiar el diseño del tablero (colores, tamaño de las celdas, fuentes) en el archivo tictactoe.py.

Personaliza los nombres de los jugadores añadiendo un cuadro de entrada para los nombres en la interfaz.

📚 Aprendizaje y Objetivos
Con este proyecto, he aprendido:

Cómo estructurar juegos simples utilizando Python.

A capturar eventos del usuario (clics) y reflejarlos en la interfaz gráfica.

Diseñar y gestionar el flujo de un juego interactivo.
